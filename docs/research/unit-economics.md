# 单位经济模型 — AI 成本、数据源、LTV/CAC

> 目标：算清"每个用户每月赚多少 / 亏多少"，作为定价与融资决策的依据。
> 所有数字基于 2026-05 公开报价 + 行业经验估算；标 [假设] 的字段访谈完后回填。

---

## 1. 模型概览

```
单用户月度毛利 = 订阅收入 - (AI 成本 + 数据源成本 + 内容安全 + 存储/带宽 + 支付通道)
单用户 LTV    = 月毛利 × 平均订阅月数 × 续费系数
CAC          = 渠道获客成本（红包、KOC、SEM、内容种草）
回本天数      = CAC / 日毛利
```

健康 SaaS 基准：**LTV/CAC ≥ 3，回本 ≤ 12 个月**。

## 2. 成本侧

### 2.1 AI 成本（最大变量）

**单次"AI 写一篇 200 字小红书正文"的 token 预算**：

| 部分 | token | 备注 |
|---|---|---|
| System prompt：垂类知识 | 1,500 | **走 cache**，命中后 90% 不计费 |
| System prompt：人设 | 500 | **走 cache** |
| System prompt：风格模板 | 300 | **走 cache** |
| 用户动态：topic + ref note 摘要 | 400 | 每次新 |
| 输出：5 个标题 + 正文 + hashtag | 600 | |
| **合计输入** | 2,300 | 其中 cache hit 2,100 |
| **合计输出** | 600 | |

**Claude Sonnet 4.6 报价（2026-05）**：

| 项 | 单价 |
|---|---|
| 输入（非缓存） | $3 / 百万 token |
| 输入（cache 命中） | $0.30 / 百万 token |
| 输入（cache 写入） | $3.75 / 百万 token |
| 输出 | $15 / 百万 token |

**单次写作成本（cache 命中场景，发生在 5min 内同一用户连续写第 2 篇起）**：

```
input cached:     2,100 × 0.30 / 1M = $0.00063
input non-cached:   200 × 3.00 / 1M = $0.0006
output:             600 × 15.0 / 1M = $0.009
─────────────────────────────────────────
合计                              ≈ $0.0102 ≈ ¥0.075
```

**单次写作成本（cache miss，每天第一次写）**：

```
input write:     2,100 × 3.75 / 1M = $0.00788
output:            600 × 15   / 1M = $0.009
─────────────────────────────────────────
合计                              ≈ $0.0169 ≈ ¥0.122
```

**混合估算**（假设博主每天写 3 篇，1 次 miss + 2 次 hit）：

```
日均成本 ≈ 0.122 + 0.075 × 2 = ¥0.27 / 用户·活跃日
```

**月成本（按月活跃 15 天）**：

| 用户类型 | 月活跃天 | 日均写作数 | 月写作数 | 月 AI 成本 |
|---|---|---|---|---|
| 个人轻度 | 8 | 1 | 8 | ¥0.85 |
| 个人 Pro | 15 | 3 | 45 | ¥4.05 |
| 团队人均 | 22 | 8 | 176 | ¥15.5 |

**另外的 AI 成本**（占总 AI 成本 ~30%）：
- 违禁词 L3 LLM 判断：用更便宜的 Haiku 4.5 模型（输入 $1 / 输出 $5）；每次 ¥0.005，量大但单价低。
- 改写工具调用：约写作量的 30%。
- 选题灵感兜底：每次 ¥0.02，低频。

**结论**：AI 成本端，个人 Pro 用户月 AI 费 ≤ ¥6；团队人均 ≤ ¥25。

## 4. LTV / CAC

### 4.1 LTV

**关键假设**：

| 套餐 | 月流失率 | 平均生命周期 | LTV |
|---|---|---|---|
| 个人 Pro | 8% | 12.5 个月 | ¥22.8 × 12.5 = ¥285 |
| 团队 Starter（团队） | 4% | 25 个月 | ¥78.4 × 4 席 × 25 = ¥7,840 |
| 团队 Pro（团队） | 3% | 33 个月 | ¥264 × 8 席 × 33 = ¥69,696 |

> 流失率估计偏乐观；个人订阅类 SaaS 经验值 5~10%；团队 2~5%。

### 4.2 CAC（按渠道）

| 渠道 | 单户 CAC | 转化率 | 备注 |
|---|---|---|---|
| 小红书自营矩阵 KOC 投放 | ¥30~80 | 5% 关注→注册，10% 注册→付费 | 主战场 |
| 公众号软文 + 投放 | ¥80~150 | 3%→付费 | |
| SEM 百度 / 微信广告 | ¥150~300 | 2%→付费 | 高 CAC |
| KOL 内容种草 | ¥50~120 | 5%→付费 | 个人博主自己来 |
| 邀请返利（老带新）| ¥10~30 | — | 团队场景特别有效 |
| MCN 直销 BD | ¥1,000~3,000 / 团队 | 30%→签约 | 高单价高毛利 |

### 4.3 LTV/CAC 表

| 套餐 | CAC | LTV | LTV/CAC | 回本天数 |
|---|---|---|---|---|
| 个人 Pro（KOC 渠道） | ¥60 | ¥285 | **4.8** ✅ | 80 天 |
| 个人 Pro（SEM）| ¥200 | ¥285 | 1.4 ❌ | 270 天 |
| 团队 Starter（BD）| ¥2,000 | ¥7,840 | **3.9** ✅ | 200 天 |
| 团队 Pro（BD）| ¥3,000 | ¥69,696 | **23** ✅✅ | 35 天 |

**结论**：
- 个人端只能走低成本内容渠道（KOC + 老带新），不能上 SEM。
- 团队 / MCN 端直销 ROI 最好，应该成为第 2 个季度的重点。

## 5. 敏感性分析

### 5.1 AI 成本翻倍场景

如果 OpenAI/Anthropic 调价或用户用量比预期高 2 倍：

| 套餐 | AI 成本 | 毛利 | 毛利率 |
|---|---|---|---|
| 个人 Pro | ¥8 | ¥18.8 | 65% |
| 团队 Starter（人均）| ¥30 | ¥63.4 | 64% |

仍然健康。**护身符**：prompt caching + 多供应商路由 + 配额限制三件套。

### 5.2 流失率翻倍场景

| 套餐 | 流失 | LTV | LTV/CAC（KOC）|
|---|---|---|---|
| 个人 Pro | 16% | ¥143 | 2.4 ⚠️ |
| 团队 Starter | 8% | ¥3,920 | 2.0 ⚠️ |

CAC 不变下 LTV/CAC 会跌破 3。**护身符**：年付折扣（锁定 12 个月）+ 团队场景靠协作粘性。

### 5.3 价格下探到 ¥19 / 月

毛利 ¥13 → LTV ¥162 → LTV/CAC（KOC）= 2.7。
**结论**：个人版价格底线 = ¥29；不要为了拉新降到 19。

## 8. 核心结论

1. **AI 成本完全可控**：Claude 4.6 + prompt caching 后单户月 AI 费 ¥4，占 ARPU 14%，毛利率 78%+。
2. **个人版定价底线 ¥29 不可破**；团队版应主推年付 + 多席位。
3. **MCN/团队 LTV/CAC 极优**（4~23 倍），应作为第 2 季度直销重点。
4. **个人端获客只能走 KOC + 老带新**，SEM ROI 不成立。
5. **¥250 万融资 + 18 个月跑通**，第 24 月达 ARR ¥4M 并实现盈利可期。
6. **最大风险**：数据源 BD 失败或被竞品挤压 → 提前签 2 家备份。
