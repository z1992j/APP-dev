```
│ └──┘└──┘└──┘└──┘└──┘            │

│                                   │
│ 正文                              │
│ ┌──────────────────────────────┐ │
│ │ 最近发现一个超绝穿搭公式🌸  │ │
│ │ 早八打工人姐妹必看…         │ │
│ │ ⚠️ "效果最好" 这词不能用     │ │ ← 违禁词内联标黄
│ └──────────────────────────────┘ │
│                                   │
│ Hashtag                           │
│ #春日穿搭 #通勤OOTD ×             │
├──────────────────────────────────┤
│ [ 检查 ] [ 预览 ] [ 排期 ] [发布] │ ← 底部操作栏
└──────────────────────────────────┘
```

**交互点**
- 图片：点 + 选择 → wx.chooseMedia → 客户端压缩 → 直传 COS → 缩略图替换占位。
- 单图编辑：裁剪 / 加水印 / 贴纸（独立编辑器 sheet）。
- 违禁词内联：实时（防抖 500ms）调 L1 词库 + 周期性触发 L2/L3。
- 自动保存：每 5s 一次 PATCH /drafts/:id；离线缓存。

## W7 · 账号档案编辑（影响 AI 效果）

```
┌──────────────────────────────────┐
│ ← 账号档案               [ 保存 ] │
├──────────────────────────────────┤
│ 头像  🖼  [ 上传 ]                │
│ 昵称  [ 小薯条                ]   │
│ 主页链接 [ xhslink.com/...     ]  │ ← 可空
│ 赛道                              │
│   [穿搭] [美妆] [母婴] [美食] +   │
│                                   │
│ ─── 人设（影响 AI 效果）────────  │
│ 性别  ⓞ女 ⓞ男 ⓞ不限              │
│ 年龄段 ⓞ18-22 ●22-28 ⓞ28-35      │
│ 城市  [ 上海      ]               │
│ 自我介绍                          │
│ [ 25 岁上海打工人，分享性价比  ] │
│ [ 通勤穿搭，预算 500 以下。   ] │ ← 多行
│                                   │
│ 我的口头禅 / 风格                 │
│ [+ 添加]                          │
│  ◆ "姐妹们集合"                  │
│  ◆ "性价比满分"                  │
│                                   │
│ 禁用词                            │
│ [+ 添加]                          │
│  ◆ 老公                          │
│  ◆ 化学                          │
└──────────────────────────────────┘
```

**为什么重要**
- 这页字段 → 直接喂 AI 写作的 `persona_block`。
- 起号期博主常常没人设，填得敷衍 → AI 写出来千篇一律。
- 引导：填一项 → 立即给"AI 试写一个标题"反馈，让用户看到价值。

## 视觉规范（一稿三定）

| 元素 | 规范 |
|---|---|

| 字体 | 系统默认（PingFang / Source Han）|
